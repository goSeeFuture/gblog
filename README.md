# gblog

本想自己搭建个博客，写写文章，但找来找去发现都比较麻烦，想到自己后期可能会改改，遂自制一个。
本人比较懒，所以博客也得符合懒人的审美，那就是必须简单。

如果你想自己搭建一个博客，不妨试试它，只需要你写Markdown文档。

特点：
    - 只要会写Markdown就能用好它
    - 没有数据库、没有javascript，只有golang
    - 提供丰富的配置，如果你修改它，欢迎入坑
    - 可以使用，但还在开发中，因为它还能更简单……
